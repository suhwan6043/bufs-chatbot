## 1. 핵심 요약 (Executive Summary)

지난주 보고 시점에서 CAMCHAT의 정답률은 **82%**였습니다.
이번 주에는 **평가 체계를 대폭 강화**(1종 → 3종, 164문항)하면서도 정답률이 **86.6%**로 향상되었습니다.

### 1.1 한눈에 보는 성과

| 구분 | 지난주 보고 | 이번주 (4/13) | 변화 |
|:---|---:|---:|:---:|
| **정답률** | 82% | **86.6%** | **+4.6%p** |
| **평가 문항 수** | 39문항 1종 | **164문항 3종** | **4.2배 확대** |
| **평가 지표** | 정답 포함 여부 1종 | 정답 포함 + 토큰 정밀도 + 오답 거부율 | **3가지로 다양화** |
| **오답 거부 성공률** | — | **100% (13/13)** | 환각 0건 |
| **추가 학습 데이터** | — | **신입생 가이드북 60페이지** (209 청크) | 신규 인제스트 |
| **퇴행(기존 정답→오답)** | — | **0건** | 안정성 확보 |

> **핵심 메시지**: 평가를 4.2배로 확대하고 지표도 3종으로 다양화했는데, 정답률이 82%에서 86.6%로 올랐습니다. 추가 학습 데이터(신입생 가이드북)를 투입하고 검색·답변 알고리즘을 4단계에 걸쳐 최적화한 결과입니다.

## 2. 이번주 추가 학습 및 개선 내용

### 2.1 추가 학습 데이터

| 항목 | 내용 |
|:---|:---|
| **신규 인제스트 문서** | 2026학년도 신입생 가이드북 (60페이지, 스캔본) |
| **OCR 처리** | Surya OCR 0.4.15를 사용하여 스캔 이미지를 텍스트로 변환 |
| **생성된 청크 수** | 209개 (검색 가능 단위로 분할) |
| **기존 청크 수** | 1,381개 → **1,590개** |
| **우선순위 설정** | 학사안내 PDF(rank 1) > 가이드북(rank 2) — 상충 시 학사안내 우선 |

> 신입생 가이드북에는 학사안내에 없는 실용 정보(캠퍼스 생활, 신입생 FAQ 등)가 포함되어 있어, 챗봇의 답변 범위가 넓어졌습니다.

### 2.2 알고리즘 개선 (4단계)

이번 주에는 검색·답변 파이프라인을 4단계에 걸쳐 체계적으로 최적화했습니다.

| 단계 | 주요 내용 | 정답률 변화 |
|:---|:---|:---:|
| **Phase 1**: 버그 수정 | 답변 검증 게이트 추가, 의미 단위 검증(AnswerUnit) 도입, 질문 분석기 엔티티 확장 | 59% → 64% |
| **Phase 2**: 검색 튜닝 | 문서 유형별 가중치 세분화(학사안내 +22%, 가이드 +18%), URL 인식 검색, TA장학 동의어 추가 | 64% → 67% |
| **Phase 3**: 데이터 보강 | 신입생 가이드북 OCR 인제스트, 문서 우선순위(source_rank) 도입, 평가 채점 정확도 개선 | 67% → 79% |
| **Phase 4**: 정밀도 향상 | 주제별 핵심 키워드 우선 배치, 조기졸업·전과 검색 기준 완화, 오답 거부 강화 | 79% → 85% |

## 3. 평가 체계 강화 — "더 다양하고 엄격하게 평가합니다"

### 3.1 왜 평가를 확대했는가

기존 39문항 1종 평가는 특정 주제에 편중되어 있어, **잘 되는 영역만 반복 측정**하는 한계가 있었습니다. 이번 주에 3개 데이터셋으로 확대하여 더 신뢰할 수 있는 성능 측정이 가능해졌습니다.

### 3.2 3개 데이터셋 구성

| 데이터셋 | 문항 수 | 특징 | 목적 |
|:---|---:|:---|:---|
| **균형 테스트셋** | 39 | 10개 학사 주제 × 3난이도, 답변불가 8문항 | 전 영역 균등 측정 |
| **학사 전문셋** | 50 | 수강신청·졸업·학사일정 심화 문항 | 핵심 업무 정확도 측정 |
| **사용자 시뮬레이션셋** | 75 | 실제 학생 질문 패턴 반영, 답변불가 5문항 | 실사용 환경 시뮬레이션 |

### 3.3 평가 지표 (1종 → 3종)

| 지표 | 의미 | 비유 |
|:---|:---|:---|
| **Contains-F1** | 정답의 핵심 정보가 답변에 포함되어 있는지 | "정답이 들어있나?" |
| **Token-F1** | 정답과 답변의 단어 수준 정밀 일치도 | "얼마나 정확하게 맞췄나?" |
| **오답 거부율** | 모르는 질문에 올바르게 거부하는 비율 | "모르면 모른다고 하는가?" |
