## 5. 잠재적 문제점 및 검토 사항

### 5.3 잠재적 문제: 실패 시나리오

만약 검색이 **실패한다면**, 다음을 확인하세요:

#### A. 필터링 조건 (student_id, doc_type)

`chroma_store.py`의 search 메서드 서명:
```python
def search(
    self,
    query: str,
    n_results: int = None,
    student_id: Optional[str] = None,      # 코호트 필터
    doc_type: Optional[Union[str, List[str]]] = None,
    semester: Optional[str] = None,
    department: Optional[str] = None,
) -> List[SearchResult]:
```

**확인사항**:
- student_id 필터가 쿼리 분석기에서 올바르게 추출되는가?
- doc_type 필터가 의도와 일치하는가?
- semester 필터가 현재 학기와 일치하는가?

#### B. 답변 생성 (Answer Generation)

검색 **자체**는 성공하지만, **답변이 생성되지 않는** 경우:
- LLM 모델 문제 (로컬 Qwen 서버 작동 확인)
- 컨텍스트 길이 초과
- 프롬프트/시스템 메시지 문제

#### C. 사용자 인터페이스 (UI)

검색과 답변은 성공하지만, UI에서 표시되지 않는 경우:
- Streamlit 렌더링 오류
- 상태 관리 문제
- 세션 변수 문제
